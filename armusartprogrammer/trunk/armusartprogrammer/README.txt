armusartprogrammer

Install:
1. Install Java runtime environment
2. Unpack the distribution ZIP file to the install directory
   (for example to C:\)
3. Follow directions in RXTX_INSTALL.txt to install the
   serial communication driver.
   On Windows this installation requires copying the attached
   rxtxSerial.dll file into the Windows\System32 directory
4. Doubleclick on armusartprogrammer-*.jar executable file to run the software
   (where * is the software version)
